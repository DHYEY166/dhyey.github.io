Autism Detection Using Deep Learning Techniques

This project aims at detecting autism using facial images. We utilized deep learning models like VGG16, VGG19, and EfficientNet to predict whether a person has autism based on their facial features. It also includes a feature where users can save their predictions for future reference.

Technologies Used: TensorFlow, Keras, OpenCV, Python, Machine Learning, Computer Vision

Details: The project was built in collaboration with fellow researchers and has shown high accuracy in predictions. The final solution is available in GitHub for anyone interested in the code and methodology used.
