Hate Speech and Offensive Language Detection

This project involves building a privacy-aware Natural Language Processing (NLP) system to detect hate speech and offensive language in social media posts. It utilizes advanced preprocessing techniques, such as tokenization, stemming, and stopword removal, and integrates word embeddings and TF-IDF for feature representation. Additionally, BERT-based Named Entity Recognition (NER) and sentiment analysis were implemented to improve classification accuracy from 82% to 95%.
