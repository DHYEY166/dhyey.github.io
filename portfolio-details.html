Autism Detection Using Deep Learning Techniques

The project aims at developing an application to detect autism using facial images as input and leveraging deep learning models such as VGG16, VGG19, and EfficientNet. The application analyzes facial features to predict the likelihood of autism. The system also includes functionality for saving predictions for future reference.

Technologies used include computer vision, deep learning, TensorFlow, and OpenCV, providing accurate results in real-time. This project was completed in collaboration with researchers from the National University of Singapore.
